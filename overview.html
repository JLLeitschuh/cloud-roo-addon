SAP HANA Cloud Addon
for Spring Roo

Intro | Tutorial | Overview

What is it about?

It's about creating Java-based web applications and to deploy and run these on a cloud platform - and all this very fast.

What technology is used?

Spring Roo is used as a Rapid Application Development (RAD) tool, which creates web applications consisting of 100% pure Java code, managed as s standard Maven project. The target platform for deployment is SAP HANA Cloud, which is a Java-based Platform-as-a-Service (PaaS) provided by SAP AG.

What kind of results do I get?

After defining a data model, Roo generates a web application consisting of a backend to handle all persistency operations, and a frontend providing a web UI for all the CRUD operations. Using the addon provided by this project you can deploy and run this application easily on SAP HANA Cloud. You can use this as a starting point for a full blown web application, or leave it as-is and use it e.g. as cloud backend for a mobile app that connects to the automatically generated REST interfaces of the backend.

What do you provide?

Software

  • Maven plugin to ease deployment of web applications on HANA Cloud
  • Roo addon to configure projects for HANA Cloud deployment, including JPA persistency
  • ZIP package of plugin/addon above (precompiled binaries) + installation instructions

Instructions

  • Installation instructions
  • Tutorial

What additional commands the plugin/addon provide?

Maven

mvn nwcloud:[deploy|start|stop|undeploy|status|apps|comps|deploy-local|start-local|stop-local|clean-local|usage|hint]

Roo

nwcloud [enable-deploy|enable-jpa|disable-deploy|disable-jpa]
