Introduction

What is it about?

In a short, it's about creating Java-based web applications and to deploy and run these on a cloud platform - and all this very fast.

How is it done?

It's done by creating the web application using the Rapid Application Development (RAD) tool Spring Roo, then using the Roo addon provided by this project to configure and deploy the web application on SAP NetWeaver Cloud. The timespan between starting with an empty directory and ending with a running web application on SAP NetWeaver Cloud should be a few minutes.

What is all this stuff?

Spring Roo is a console-based Rapid Application Development tool maintained by SpringSource. Roo generates 100% pure Java code, managed as a standard Maven project that can easily be imported into an IDE like Eclipse. Roo can be extended by addons, that provide additional functionality to the tool.

The SAP NetWeaver Cloud Addon for Spring Roo is an addon for Roo which adds commands that make it easy and comfortable to configure your Roo-generated projects for use on SAP NetWeaver Cloud and to do the actual deployment.

SAP NetWeaver Cloud is a Java-based Platform-as-a-Service (PaaS) provided by SAP AG. Currently SAP AG provides a free 90-day-trial of SAP NetWeaver Cloud, so it's easy to give it a try.

Try it!

If you want to give it a try, just download the addon package and follow the tutorial we prepared. The tutorial will show you how to create a web application, as well as how to deploy it and start it in the SAP NetWeaver Cloud - and all this in just a few minutes.
